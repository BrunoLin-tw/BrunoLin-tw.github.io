OpenAI 這次推出 GPT-5.5，我第一眼看的其實不是 benchmark，而是價目表。

![GPT-5.5 踏上排行榜第一名、超越 Opus 4.7 的科技風格插圖](/assets/images/posts/gpt55-vs-opus47-ranking-podium.png)

API 定價直接開到 input 5 美元、output 30 美元。這個數字很直接，因為 output 已經高過 Claude Opus 4.7 的 25 美元。這幾年很少看到 OpenAI 在旗艦模型的輸出定價上壓過 Anthropic，現在它敢這樣開，代表它對高端市場的位置很有把握。

根據 OpenAI 的[官方發布](https://openai.com/index/introducing-gpt-5-5/)，GPT-5.5 在 FrontierMath Tier 4 來到 39.5%，也在多項資安測試裡用更少嘗試次數解掉複雜挑戰。這些數字背後的意思，其實比排行榜本身更重要。模型不只更會答題，也更能在高難任務裡持續推進，少走歪路，少中途停住。

這對 agent workflow 很關鍵。現在真正貴的，常常不是 token，而是重試、返工、人工接手和上下文浪費。模型單價高一點，如果能少救火幾次，整體成本未必比較高。從這個角度看，OpenAI 這次是在替「可靠完成複雜工作」直接開價。

另一個值得補的點，是 GPT-5.5 也讓外界更自然把它跟先前流出的代號 Spud 連在一起看。從這次產品定位來看，這代很明顯是朝更能進工作流的方向推，不只是秀能力，而是想把更多真實任務吃下來。

這裡也剛好可以對照 Anthropic 最近的動作。

前陣子 Anthropic 先用 Mythos 把高階資安模型的話題拉走，強調 limited preview、關鍵基礎設施、防禦型資安，成功把自己放到一個高風險也高價值的位置。OpenAI 這次推 GPT-5.5，某種程度上就是正面回應這條線：你有 Mythos，我這邊也有更全面商用、也更敢開價的旗艦模型，市場焦點不可能只留在你那邊。

而且競爭壓力很快就反映在產品上。GPT-5.5 發布後，Anthropic 很快重置 Claude 使用額度，也開始修補被用戶抱怨很久的 Opus 降智問題。這種事平常都修得慢，一有像樣的對手施壓，速度就恢復正常。說穿了，不是以前修不好，是以前還不夠痛。

所以我自己的看法很簡單：GPT-5.5 這次真正重要的地方，是 OpenAI 開始把高端模型當高端服務賣，而且價格也不再客氣。Anthropic 則用 Mythos 守住高風險、高價值場景的話語權，再用產品修補回應市場壓力。

接下來比的，已經不只是誰更聰明，而是誰更值得被放進真實工作流裡。

這才是高端模型最值錢的地方。
