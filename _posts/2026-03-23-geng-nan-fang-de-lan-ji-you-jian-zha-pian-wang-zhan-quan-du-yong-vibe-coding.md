最近 Hacker News 上「They’re vibe-coding spam now」這篇挺熱門，104 points，62 則留言。文章講，垃圾郵件附的詐騙網站以前手工亂搞，現在全靠 AI 「感覺式編碼」——vibe-coding——隨便 prompt 一堆出來，結構安全不管，感覺對了就上。

用 Cursor、Claude Code 這些工具，懶得按規矩，結果 SQL 注入、資料明文外洩，黑客一看就破。

過去 spam 低質是故意的，專過濾笨蛋；AI 來了，海量生產/以假亂真讓人防不勝防。
